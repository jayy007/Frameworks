Name of the field which value will be recalculated.

fdeps: list of str

Dependen fields.

start_byte: str

Byte in which the value of the main field begins. Must be a str. Fields must be preceded by ‘this’. It can be a complex expression: ‘70 - this.field_len’

expression: str

Expression with which the new field value will be calculated dynamically. Fields must be preceded by ‘this’. Can be complex: ‘this.field_len - this.field2_len - 2’
